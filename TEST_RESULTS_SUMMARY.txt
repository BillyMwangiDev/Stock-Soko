================================================================================
                          TROUBLESHOOTING
================================================================================

If backend not responding:
  1. Check process: netstat -ano | findstr ":8000"
  2. Restart: cd backend && python -m uvicorn app.main:app --reload

If frontend won't load:
  1. Check QR code in terminal
  2. Verify backend at http://localhost:8000/docs
  3. Update frontend/.env.local with your IP
  4. Press 'r' in Expo terminal to reload

If database issues:
  1. Check stocksoko.db exists
  2. Run: python manage.py init-db
  3. Reseed: python seed_database.py
